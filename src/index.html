URL params examples

  • active-section
    Bold navigation item when element is in view
  • heading=On+this+page
    Customize navigation title
  • auto-close
    Close by clicking outside of menu
  • z-index=5000
    Custom z-index
  • css=external
    Use external stylesheet instead of built in css
  • smooth
    Smooth scrolling
  • top=200px
    Customize top value
  • nest
    Nest navigation elements into child lists
  • search
    Enable a search input featuring autocomplete
  • show-search-after
    Search will appear after a set number of items appear in the nav
  • search-focus
    When maximizing the nav, the search input will be focused automatically
